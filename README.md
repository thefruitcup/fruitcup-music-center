# Fruitup Music Center
A Windows Media Center Inspired Music Player created in Godot 4

**This program is still in an Alpha State! Please report any issues you face!**

### Minimal System Requirements
<ul>
  <li>Windows 10+ 64 bit</li>
  <li>Any GPU that runs Direct3D 12 or has full Vulkan Support</li>
  <li>430+ MB of RAM (Tested with 1184 Music Tracks)</li>
  <li>180 MB of Storage</li>
</ul>

*Linux Requirements are most likely the same.*

Linux Support will most likely be limited as I do not have Linux, but the program should run fine on it.

### Features

<ul>
  <li>Audio Controls (No Repeat)</li>
  <li>Directory Addition & Removal</li>
  <li>Music Information</li>
  <li>MP3, WAV & OGG Support</li>
</ul>

# How to Compile
First you'll need to download Godot Engine v4.6.1 Mono:<br>
https://godotengine.org/download/archive/4.6.1-stable/

Clone the project:<br>
```git clone https://github.com/thefruitcup/fruitcup-music-center.git```

Open Godot to the Project and once everything is done importing, click on ```Project``` at the top and ```Export```.<br>
If you haven't used Godot before, you will have to download the Binaries to be able to export the project.

Once that's done, go back to the Export Window, click on ```Export Project``` and select where you'd like to export it.<br>
Now you have compiled Fruitcup Music Center!
